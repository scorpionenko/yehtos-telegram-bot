# ЄХтось

Telegram-бот для анонімної психологічної підтримки, де користувачі можуть допомагати один одному через анонімний чат.

---

# Про проєкт

**ЄХтось** — це Telegram-бот, який надає можливість користувачам отримувати емоційну підтримку від інших людей у безпечному та анонімному середовищі.

Користувач може описати свою проблему або емоційний стан, після чого бот передає це повідомлення іншому користувачу, який готовий допомогти. Після завершення розмови отримувач підтримки може оцінити допомогу за п’ятибальною шкалою.

Таким чином формується система рейтингу, яка дозволяє визначити користувачів, що надають найбільш якісну підтримку.

Основна ідея сервісу полягає у створенні спільноти, де люди можуть підтримувати одне одного у складні моменти.

---

# Які проблеми вирішує бот?

### Доступність психологічної підтримки

Не всі люди мають можливість звернутися до професійного психолога. Бот дозволяє швидко отримати підтримку від інших людей.

---

### Анонімність

Користувачі не бачать особистих даних один одного. Вся комунікація проходить через бота.

---

### Швидкість взаємодії

Повідомлення миттєво передаються іншим користувачам, які готові допомогти.

---

### Соціальна підтримка

Користувачі можуть не тільки отримувати допомогу, а й підтримувати інших людей.

---

# Основний тип користувача

Бот орієнтований на людей, які:

* переживають стрес або складний емоційний період
* потребують можливості поговорити з кимось анонімно
* хочуть допомагати іншим людям
* шукають безпечне середовище для спілкування

---

# Основний функціонал

* створення запиту на емоційну підтримку
* анонімний чат між користувачами
* можливість допомагати іншим людям
* система оцінювання допомоги
* формування рейтингу користувачів
* перегляд особистого профілю та статистики

---

# Як працює бот

1. Користувач запускає бота через команду **/start**
2. Користувач може створити запит на психологічну підтримку
3. Запит додається до черги
4. Інший користувач може взяти запит і почати анонімний діалог
5. Після завершення розмови користувач оцінює допомогу
6. Система оновлює рейтинг користувача

---

# Технології

* **Мова програмування:** Python
* **Telegram бібліотека:** aiogram
* **База даних:** PostgreSQL
* **Інструмент адміністрування БД:** pgAdmin
* **Платформа:** Telegram Bot API

---

# Архітектура системи

Система побудована за клієнт-серверною архітектурою.

Користувач взаємодіє з ботом через Telegram. Повідомлення передаються через Telegram Bot API до серверної частини, яка обробляє логіку роботи та зберігає дані у базі.

Схема взаємодії:

```
User
↓
Telegram
↓
Telegram Bot API
↓
Python Bot (aiogram)
↓
PostgreSQL Database
```

# Аналіз конкурентів

Існує декілька платформ для онлайн психологічної підтримки.

Однак більшість із них або не забезпечують повної анонімності, або не орієнтовані на швидку взаємодію між користувачами.
